◆ find_files()

std::set< fs::path > FileManager::find_files ( const std::string_view &  regex)
static

Regex file name search.

Regex is split into groups by PATH_SEPARATOR. those groups are searched in the given path. If group starts with '?', it is treated as optional.
example:
/home/user/?.+/.+\.txt
will match:
Any file in /home/user/(any directory)/(any directory) with extension .txt
or any file in /home/user/(any directory) with extension .txt
like:
/home/user/test/test.txt
/home/user/test/test2/test.txt
/home/user/test/test2/test2.txt
The regex matching is case sensitive.

Parameters
regexto be matched against file paths.
Returns
set of file paths matching the regex.
