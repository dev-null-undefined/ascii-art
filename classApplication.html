Detailed Description

The Application class controls ncurses and menus.

Application is responsible for starting and stopping ncurses, and handling basic input keys. It also handles argument parsing and data source loading. And manages menus and their display. It is not multi-threaded, but it does simulate a multi-threaded like environment using timeout on blocking ncurses calls.
