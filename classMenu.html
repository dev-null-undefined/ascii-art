Handle key input.

This method handles any scene specific key shortcuts. If shortcut was handled it will set the handled bit to true. If the scene also needs redraw after the key stroke it will return true. Returning true also means the key was handled automatically.

Parameters
inputKey code that was pressed.
[out]handledout parameter signalizing if the key was handled by the scene.
Returns
True if update/redraw is needed, otherwise false.
