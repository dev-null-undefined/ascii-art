Detailed Description

Settings class.

class for storing, parsing, managing and loading settings.

Constructor & Destructor Documentation

◆ Settings()

Settings::Settings ( )

Settings constructor.

constructors Settings class with default values and tries to load settings from files based on the CONFIG_FILE_PATHS. Reading in reversed order to allow overriding of settings from files with higher priority. If key provide multiple times, the last value is used.
